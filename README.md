# Elegant Clock

Elegant Clock 是一个面向 Windows 与 Linux 的可自定义桌面时钟软件，提供日期显示、透明显示、倒计时、正计时、番茄钟和日历式提醒功能。

## 功能

- 默认只显示时钟，设置用右下角齿轮打开独立窗口，其他功能用右下角三横杠打开独立窗口
- 透明/不透明显示切换，可调整透明度
- 字体、时钟字号、字体颜色与背景颜色自定义
- 当前时间与日期显示
- 按小时/分钟/秒设置倒计时
- 倒计时到指定日期时间
- 倒计时运行时会以较小字体同步显示在主时钟内，控制仍保留在功能窗口中
- 正计时/秒表：开始、暂停、重置
- 番茄钟：专注、短休息、长休息与长休间隔配置
- 提醒：按指定日期时间添加提醒，支持标题、备注、本地保存和系统通知
- 铃声：默认内置 `ringtone_default.mp3`，也可自行选择本地音频作为闹钟铃声，支持试听与停止试听
- 开机自启动：安装时可选择配置，安装后也可在设置中再次调整
- 无边框窗口：隐藏到状态栏、最小化、最大化/还原、关闭、可选置顶、托盘后台运行
- 闲置紧凑模式：5 秒无操作后隐藏窗口控件，仅保留圆角时钟/日期视图；点击恢复完整窗口，拖动只移动位置
- 独立设置窗口与功能窗口不受 5 秒闲置紧凑模式影响；在功能窗口中选择功能后会在同一窗口打开，并提供左上角返回
- 单实例运行：重复启动时自动显示已运行窗口，不再新开进程窗口

## 开发运行

```bash
npm install
npm start
```

## 本地打包

```bash
npm run dist:win
npm run dist:linux
npm run package:arch
```

Windows 目标产物：`.exe`、`.msi`。

Linux 目标产物：`.deb`、`.dpkg`、`.rpm`、`.pkg.tar.zst`。

每个 Release 会额外上传 `SHA256SUMS.txt`，用于校验所有安装包资产。
